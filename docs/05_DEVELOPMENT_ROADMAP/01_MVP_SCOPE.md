# NEXORA AI

## MVP Scope Document

Version: 1.0
Status: Development Planning Foundation

# 1. Introduction

The Minimum Viable Product (MVP) defines the first functional version of NEXORA AI.

The purpose of the MVP is to validate the core idea:

"Can AI help students understand their skills and connect with better career opportunities?"

The MVP focuses on:

* Student onboarding
* Resume intelligence
* Skill analysis
* Basic job matching
* Recruiter workflow

# 2. MVP Goals

The first version should prove:

## Student Value

Students can:

* Create career profiles
* Upload resumes
* Understand skill gaps
* Receive AI recommendations

## Recruiter Value

Recruiters can:

* Create company profiles
* Post jobs
* Find suitable candidates

## Platform Value

NEXORA AI can:

* Analyze career data
* Generate intelligent insights
* Match talent with opportunities

# 3. MVP User Roles

## Student

Main user of the platform.

Capabilities:

* Register account
* Create profile
* Upload resume
* Add skills
* View AI analysis
* Explore opportunities

## Recruiter

Hiring-side user.

Capabilities:

* Create company profile
* Create job posts
* View candidates
* Review matches

## Admin

Platform management.

Capabilities:

* Manage users
* Monitor platform
* Handle reports

# 5. Features Not Included in MVP

Advanced features are postponed:

* AI Interview Agent
* Voice Assistant
* Full GitHub analysis
* Predictive placement engine
* Global talent graph
* Enterprise HR integration

Reason:

Focus on validating the core product first.
