# 1. Introduction

The AI Engine is the central intelligence layer of NEXORA AI.

It transforms raw user data into meaningful career intelligence.

The AI Engine is responsible for:

- Understanding skills
- Analyzing resumes
- Evaluating projects
- Matching candidates
- Generating recommendations
- Supporting career decisions
